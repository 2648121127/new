$$参数：$$(选择元素，父级元素)  注意：查找到的都是数组，所以有map方法
实现下面方法的地址：https://pvp.qq.com/m/m201706/index.shtml  
（1）$$('.hero-nav > li').map((li,i) => {
    return {
        name:li.innerText,
    }
})
（2）$$('.hero-nav > li').map((li,i) => {
    return {
        name:li.innerText,
        heroes:$$('li',$$('.hero-list')[i]).map(el=>{
            return {
                name:$$('h3',el)[0].innerHTML,
                avatar:$$('img',el)[0].src
            }
        })
    }
})
